# PowerProfile Apps

Use as ferramentas disponibilizadas pelo servidor MCP `powerprofile-apps`.

- Pesquise e habilite somente os aplicativos relevantes ao pedido atual.
- Quando uma conta ainda nao estiver conectada, apresente o link OAuth retornado pelo servidor e aguarde o usuario concluir.
- Nunca solicite senhas, tokens ou codigos de autenticacao no chat.
- Antes de enviar mensagens, publicar conteudo, excluir dados, alterar campanhas ou realizar compras, solicite confirmacao do usuario.
- Prefira operacoes de leitura quando o pedido nao autorizar explicitamente uma alteracao externa.
