# Gmail

Use the managed MCP server `PowerProfile Apps` and constrain tool discovery to toolkit `gmail`.

If the account is not connected, use connection management and present its secure authorization link. Load only the schemas needed for the request. Prefer read-only actions; confirm before sending, deleting, or changing messages.
