# Google Agenda

Use the managed MCP server `PowerProfile Apps` and constrain tool discovery to toolkit `googlecalendar`.

If the account is not connected, use connection management and present its secure authorization link. Load only the schemas needed for the request. Confirm before creating, changing, inviting attendees to, or deleting events.
