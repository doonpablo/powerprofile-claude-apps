# Follow Up Boss

Use the managed MCP server `PowerProfile Apps`.

1. Search for tools relevant to the user's request and constrain discovery to toolkit `follow_up_boss`.
2. If the account is not connected, use the connection-management tool, present the returned secure link, and wait for completion.
3. Load only the schemas needed for the current task.
4. Prefer read-only operations unless the user explicitly requests a change.
5. Before sending messages, publishing content, deleting data, changing campaigns, making purchases, or performing another consequential external action, ask for confirmation.
