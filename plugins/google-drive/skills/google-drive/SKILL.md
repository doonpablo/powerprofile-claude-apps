# Google Drive

Use the managed MCP server `PowerProfile Apps` and constrain tool discovery to toolkit `googledrive`.

If the account is not connected, use connection management and present its secure authorization link. Load only the schemas needed for the request. Creation and upload are supported; confirm before deleting, moving, overwriting, or changing sharing permissions.
